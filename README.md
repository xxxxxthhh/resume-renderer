# Resume Renderer

AI-assisted Markdown-to-PDF workflow for polished application materials.

This is a standalone project extracted from an application-materials workflow. The goal is to keep source documents in Markdown, use AI to review them against a job description, and produce elegant recruiter-ready PDFs through a repeatable, reviewable, open-source toolchain.

Current status: design brief only. See [docs/design.md](docs/design.md).
